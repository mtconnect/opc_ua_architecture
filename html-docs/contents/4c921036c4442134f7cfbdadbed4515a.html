deriveEURange

UMLOperation
MTConnect :: MTConnectDevices :: Data Items :: «mixin»MTNumericDataItemType :: deriveEURange

Description

Uses the MTConnect Constraints element if present to derive the minimum and maximum values for the numeric values. This applies to both the Numeric Event and the Sample types.

Specification

EURange.High <- self.Constraints.Maximum
