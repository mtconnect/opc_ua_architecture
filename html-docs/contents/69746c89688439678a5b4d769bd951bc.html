DataItemTypeFactory

UMLClass ««Type Factory»Type Factory»
MTConnect :: MTConnectDevices :: Factories :: «Type Factory»DataItemTypeFactory

Description

Based on the data item category, type, and subType, this class creates a new OPC/UA type and also provides the template parameter for the ParentType from which this type is derived.

See the Data Item Type Factory.

Generalized Elements

  • «Type Factory»TypeFactory

Operations

Visibility Name Description
public deriveParentType(type, subType)

The parent type is derived from the category as follows:

  • SAMPLE -> SampleType
  • EVENT ->
    • Enumerated Value -> MTEnumeratedEventType
    • Integer Value -> MTNumericEventType
    • Otherwise -> MTStringEventType
public deriveTypeName(type, subType)

Used to derive the class name for creating a pascal case name from the sub type and the type. For example type ROTARY_VELOCITY and subType ACTUAL will become ActualRotaryVelocity.
